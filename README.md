## 🛠️ Stack Tecnológico

- **Frontend**: Svelte + SvelteKit
- **Estilo**: 98.css (estética retro)
- **Build**: Vite
- **Deploy**: GitHub Pages

---

## 🚀 Ejecutar Localmente

**Requisitos**: Node.js 18+

```bash
git clone https://github.com/Quirogama/Quirogama.github.io.git
npm install
npm run dev
```

Opcional: crea `.env` desde `.env.example` para configurar `VITE_FORMSPREE_ID` sin editar código.

## ✅ Tests

```bash
npm run test
```

Modo interactivo (re-ejecuta al guardar):

```bash
npm run test:watch
```
